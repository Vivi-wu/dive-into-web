## 路由

react-router-dom 的 Hooks 中的 `useHistory` 只能用在函数式组件里.

使用底层 `<Router>` 最常见的情况是将自定义历史记录与状态管理库（如Redux或Mobx）进行同步。但是 React Router 并不是一定要根状态管理库一起使用，它仅用于深度集成 history。
