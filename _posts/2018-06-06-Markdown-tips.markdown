整理 .markdown 常用语法。


## Markdown

### 在新 tab 页打开链接

`[]` 里是链接文案，`()` 里是链接地址，`{}` 里是元素属性。

```
[这个插件](https://github.com/bfred-it/iphone-inline-video){:target="_blank"}
```

## Visual Studio Code

### 在指定目录下的文件中查找

在订单列表目录下的 .vue 文件里查找

	**/sales-order/**/*.vue

### 跳转至函数定义处

鼠标悬停在函数名上，按照 `Ctrl` 键，当出现超链接下划线时，点击可（打开对应文件并）跳转至函数定义处。
